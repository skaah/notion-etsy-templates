## 💡 TIPS UTILISATION

### Dans Canva:
1. Ouvrez **"Magic Media"** (icône étoile ✨)
2. Collez le prompt (anglais = meilleurs résultats)
3. Cliquez **Générer**
4. Sélectionnez le meilleur résultat
5. Ajoutez le texte manuellement après

### Optimisation:
- Générez 4 variantes par prompt
- Choisissez la meilleure
- Modifiez couleurs si besoin (Éditer → Ajuster)
- Ajoutez texte en dernier

### Format:
- Ratio: **1:1** (carré) pour Hero/Bundle
- Ratio: **3:2** pour les autres
- Style: **Photo** ou **Graphique** selon besoin
