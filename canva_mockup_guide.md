## 🎯 ASTUCES PRO CANVA

### 1. Alignement parfait
- Utilisez la **grille** (Affichage → Afficher la grille)
- Activez les **lignes guides** (s'affichent automatiquement)

### 2. Cohérence des couleurs
- Créez une **palette de marque** (Menu → Palette de marque)
- Sauvegardez les 3 couleurs de votre drapeau
- Réutilisez-les sur toutes les images

### 3. Polices recommandées
- **Titres:** Montserrat Bold, Poppins Bold, Bebas Neue
- **Texte:** Open Sans, Lato, Roboto
- **Élégant:** Playfair Display, Merriweather

### 4. Icônes gratuites
- Onglet **"Éléments"** → Rechercher
- Mots-clés: "icon", "flat icon", "outline"
- Éviter les éléments "Pro" (avec couronne)

### 5. Export optimal
- Format: **PNG** (meilleure qualité)
- Taille: **Réduire la taille du fichier** (pour Etsy)
- Ou **JPG** 80% qualité (fichiers plus petits)

## 🖼️ MÉTHODE: CAPTURE NOTION SIMPLIFIÉE

Si vous voulez du contenu réel dans vos mockups:

### Option A: Capture partielle (Recommandée)
1. Ouvrez votre template Notion
2. Utilisez **Snipping Tool** (Windows) ou **Cmd+Shift+4** (Mac)
3. Capturez juste une zone (ex: une base de données)
4. Importez dans Canva
5. Glissez dans le mockup laptop

### Option B: Export Notion
1. Dans Notion: **...** → **Exporter** → **PDF**
2. Convertissez PDF → image (ilovepdf.com)
3. Utilisez dans Canva

### Option C: Créer de zéro (Plus pro)
1. Recréez les tableaux directement dans Canva
2. Utilisez les éléments "Tableau" ou formes
3. Plus beau et plus contrôlable !
