
**Introduction:**
```
Organisez tous vos cours (CM, TD, TP) en un seul endroit. 
Visualisez votre semaine d'un coup d'œil et ne manquez plus aucun cours !
```

**Instructions:**
```
🎯 Comment utiliser cette base :

1. Cliquez sur "Nouveau" pour ajouter un cours
2. Remplissez les informations :
   • Nom du cours (ex: Mathématiques Analyse)
   • Jour de la semaine
   • Heure de début et fin
   • Salle ou amphithéâtre
   • Nom du professeur
   • Type : CM, TD ou TP

3. Utilisez les vues :
   • Vue Tableau : Tous les cours
   • Vue Calendrier : Par semaine
   • Filtre par jour : Une journée spécifique

📌 Exemple de cours à ajouter :
• Mathématiques | Lundi 9h-11h | Amphi A | Prof. Dupont | CM
• Physique TP | Mardi 14h-16h | Labo B | Prof. Martin | TP
• Informatique | Mercredi 10h-12h | Salle 203 | Prof. Bernard | TD
```

### Comment copier ce contenu dans Notion :

3. **Copiez** (Ctrl+C / Cmd+C)
4. **Collez** dans Notion (Ctrl+V / Cmd+V)

6. **Ajustez** les titres et la mise en page si besoin

Créez une page "Modèle" avec tout le contenu bien formaté, puis **dupliquez-la** pour chaque langue au lieu de tout recopier !
